# Signal Processing

Signal processing focuses on the analysis, modification, and synthesis of signals such as sound, images, and scientific measurements.

## Topics

- Networking
- Low Pass Filters
- High Pass Filters
- Butterworth Filter
- Filter Classes
- Chebyshev Filters
- Discrete-time Signal Processing
- Sampling Theorem
- Discrete Fourier Transform
- Fast Fourier Transform
- Spectral Leakage
- z-Transform
- Adaptive Filtering
- Kalman Filtering
- Least Mean Squares Algorithm
- Recursive Least Squares Algorithm
- Affine Projection Algorithm
- Lattice Filtering
- Voltera Filters
- Laplace Transforms
- Transfer Functions
- Modulation
- Parseval's Theorem
- Correlation Filtering
- Convolution
