# Álgebra Abstracta: Fundamentos de la Teoría de Grupos

Este capítulo introduce los conceptos fundamentales de la teoría de grupos, una de las estructuras algebraicas más importantes en matemáticas. Exploraremos la definición de grupos, examinaremos propiedades clave y trabajaremos con ejemplos concretos.

## ¿Qué es un Grupo?

Un **grupo** es un conjunto $G$ junto con una operación binaria $\star$ que satisface cuatro propiedades clave:

1. **Clausura**: Para todo $a, b \in G$, tenemos $a \star b \in G$
2. **Asociatividad**: Para todo $a, b, c \in G$, tenemos $(a \star b) \star c = a \star (b \star c)$
3. **Elemento Identidad**: Existe un elemento $e \in G$ tal que $e \star a = a \star e = a$ para todo $a \in G$
4. **Elementos Inversos**: Para cada $a \in G$, existe un elemento $a^{-1} \in G$ tal que $a \star a^{-1} = a^{-1} \star a = e$

## Ejemplos de Grupos

### Los Enteros bajo la Adición
El conjunto $\mathbb{Z}$ con la operación de adición forma un grupo:
- **Clausura**: La suma de cualquier par de enteros es un entero
- **Asociatividad**: $(a + b) + c = a + (b + c)$ para todos los enteros $a, b, c$
- **Identidad**: $0$ es la identidad aditiva
- **Inversos**: Para cualquier entero $a$, su inverso aditivo es $-a$

### Grupos Simétricos
El conjunto de todas las permutaciones de $n$ elementos forma el grupo simétrico $S_n$ bajo composición de funciones.
