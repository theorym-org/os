# 第一章：群

*群*的概念是整个数学中最深刻、影响最深远的思想之一。群论诞生于十九世纪初对多项式方程的研究——通过埃瓦里斯特·伽罗瓦的革命性工作——此后渗透到数学和物理学的几乎每一个分支。从几何图形的对称性到基本粒子的结构，从密码学协议到曲面的分类，群提供了我们理解对称性和结构的语言。

本章从第一性原理发展群论。我们从定义群的公理开始，然后建立所有群共有的基本性质，最后探索激发和阐明抽象理论的丰富例子。

## 1.1 二元运算

在定义群之前，我们必须首先明确什么是集合上的*运算*。数的加法和乘法是熟悉的原型，但抽象概念要广泛得多。

<Definition number="1.1.1" name="二元运算">
设 $S$ 是非空集合。$S$ 上的**二元运算**是一个函数
$$\star : S \times S \to S$$
它将 $S$ 的每个有序对 $(a, b)$ 映射到 $S$ 的唯一元素 $a \star b$。
</Definition>

对所有 $a, b \in S$，要求 $a \star b \in S$ 称为**封闭性**。这不是额外的公理，而是作为 $S$ 上二元运算的定义的一部分。当我们写 $\star : S \times S \to S$ 时，我们断言该运算取 $S$ 的任意两个元素并产生 $S$ 的一个元素。

<Example number="1.1.2" name="标准运算">
以下是二元运算：
1. $\mathbb{Z}$ 上的加法：对任意 $a, b \in \mathbb{Z}$，有 $a + b \in \mathbb{Z}$
2. $\mathbb{R}$ 上的乘法：对任意 $a, b \in \mathbb{R}$，有 $a \cdot b \in \mathbb{R}$
3. 函数复合：若 $f, g : X \to X$，则 $f \circ g : X \to X$
</Example>

<Example number="1.1.3" name="反例">
以下不是二元运算：
1. $\mathbb{R}$ 上的除法：当 $b = 0$ 时 $a / b$ 无定义
2. $\mathbb{N}$ 上的减法：$2 - 5 = -3 \notin \mathbb{N}$（封闭性失效）
</Example>

<Definition number="1.1.4" name="二元运算的性质">
设 $\star$ 是集合 $S$ 上的二元运算。我们说 $\star$ 是：

1. **结合的**，若对所有 $a, b, c \in S$，$(a \star b) \star c = a \star (b \star c)$
2. **交换的**，若对所有 $a, b \in S$，$a \star b = b \star a$
</Definition>

<Remark>
结合律允许我们不加括号地写 $a \star b \star c$。交换律虽然是一个好性质，但对我们的大部分理论*不是*必需的——事实上，一些最重要的群是非交换的。
</Remark>

## 1.2 群的定义

我们现在来到本章的核心定义。群是配备了满足三个基本公理的二元运算的集合。

<Definition number="1.2.1" name="群">
**群**是有序对 $(G, \star)$，其中 $G$ 是非空集合，$\star$ 是 $G$ 上满足以下公理的二元运算：

**(G1) 结合律。** 对所有 $a, b, c \in G$：
$$(a \star b) \star c = a \star (b \star c)$$

**(G2) 单位元。** 存在元素 $e \in G$ 使得对所有 $a \in G$：
$$e \star a = a \star e = a$$

**(G3) 逆元。** 对每个 $a \in G$，存在元素 $a^{-1} \in G$ 使得：
$$a \star a^{-1} = a^{-1} \star a = e$$
</Definition>

<Remark>
几点说明：

1. 公理按一般性递减的顺序列出。许多代数结构只满足 (G1)（半群）或 (G1) 和 (G2)（幺半群）。群需要全部三条。

2. 封闭性隐含在 $\star$ 是 $G$ 上二元运算的陈述中。

3. 单位元公理要求 $e$ 是*双边*单位元：它必须在左边和右边都起作用。类似地，逆元公理要求*双边*逆元。
</Remark>

<Definition number="1.2.2" name="阿贝尔群">
群 $(G, \star)$ 称为**阿贝尔群**（或**交换群**），若对所有 $a, b \in G$：
$$a \star b = b \star a$$
这个术语是为了纪念尼尔斯·亨利克·阿贝尔（1802–1829），他对方程理论做出了基础性贡献。
</Definition>

<Definition number="1.2.3" name="群的阶">
群 $G$ 的**阶**，记作 $|G|$，是底层集合的基数。若 $|G|$ 有限，我们说 $G$ 是**有限群**；否则，$G$ 是**无限群**。
</Definition>

## 1.3 基本性质

群的公理虽然简洁，却有强大的推论。我们现在建立在*每个*群中都成立的几个基本性质。

<Theorem number="1.3.1" name="单位元的唯一性">
在任意群 $(G, \star)$ 中，单位元是唯一的。
</Theorem>

<Proof>
设 $e$ 和 $e'$ 都是 $G$ 中的单位元。我们证明 $e = e'$。

因为 $e$ 是单位元，有 $e \star e' = e'$。

因为 $e'$ 是单位元，有 $e \star e' = e$。

因此 $e = e \star e' = e'$，证毕。
</Proof>

<Theorem number="1.3.2" name="逆元的唯一性">
在任意群 $(G, \star)$ 中，每个元素有唯一的逆元。
</Theorem>

<Proof>
设 $a \in G$，假设 $b$ 和 $c$ 都是 $a$ 的逆元。则：
$$b = b \star e = b \star (a \star c) = (b \star a) \star c = e \star c = c$$
其中我们使用了结合律和逆元的定义性质。
</Proof>

逆元的唯一性证明了我们用 $a^{-1}$ 表示 $a$ 的*唯一*逆元的记法是合理的。

<Theorem number="1.3.3" name="消去律">
设 $(G, \star)$ 是群，$a, b, c \in G$。则：
1. **左消去律：** 若 $a \star b = a \star c$，则 $b = c$
2. **右消去律：** 若 $b \star a = c \star a$，则 $b = c$
</Theorem>

<Proof>
我们证明左消去律；右消去律类似。

设 $a \star b = a \star c$。两边左乘 $a^{-1}$：
$$a^{-1} \star (a \star b) = a^{-1} \star (a \star c)$$
由结合律：
$$(a^{-1} \star a) \star b = (a^{-1} \star a) \star c$$
因为 $a^{-1} \star a = e$：
$$e \star b = e \star c$$
因此 $b = c$。
</Proof>

<Theorem number="1.3.4" name="乘积的逆">
设 $(G, \star)$ 是群，$a, b \in G$。则：
$$(a \star b)^{-1} = b^{-1} \star a^{-1}$$
</Theorem>

<Proof>
我们验证 $b^{-1} \star a^{-1}$ 满足 $a \star b$ 的逆元的定义性质：
$$(a \star b) \star (b^{-1} \star a^{-1}) = a \star (b \star b^{-1}) \star a^{-1} = a \star e \star a^{-1} = a \star a^{-1} = e$$
类似地，$(b^{-1} \star a^{-1}) \star (a \star b) = e$。由逆元的唯一性，$(a \star b)^{-1} = b^{-1} \star a^{-1}$。
</Proof>

<Remark>
公式 $(ab)^{-1} = b^{-1}a^{-1}$ 有时称为"袜子和鞋子"法则：要撤销先穿袜子再穿鞋子的动作，你必须先脱鞋子，再脱袜子。顺序是颠倒的。
</Remark>

<Corollary number="1.3.5">
对群 $G$ 中的任意元素 $a$：$(a^{-1})^{-1} = a$。
</Corollary>

<Proof>
元素 $a$ 满足 $a^{-1} \star a = a \star a^{-1} = e$，这正是 $a$ 是 $a^{-1}$ 的逆元的条件。
</Proof>

## 1.4 群的例子

群的抽象定义涵盖了大量的数学结构。我们现在考察最重要的例子，它们将作为我们整个研究的参照。

### 数的加法群

<Example number="1.4.1" name="整数">
整数集 $\mathbb{Z}$ 在加法下构成群，记作 $(\mathbb{Z}, +)$：
- **封闭性：** 两个整数的和是整数
- **结合律：** 对所有 $a, b, c \in \mathbb{Z}$，$(a + b) + c = a + (b + c)$
- **单位元：** 整数 $0$ 满足 $0 + a = a + 0 = a$
- **逆元：** 对每个 $a \in \mathbb{Z}$，整数 $-a$ 满足 $a + (-a) = 0$

这个群是阿贝尔群且是无限群。
</Example>

<Example number="1.4.2" name="有理数、实数和复数">
类似地，$(\mathbb{Q}, +)$、$(\mathbb{R}, +)$ 和 $(\mathbb{C}, +)$ 都是加法下的阿贝尔群。
</Example>

### 乘法群

<Example number="1.4.3" name="非零有理数">
集合 $\mathbb{Q}^* = \mathbb{Q} \setminus \{0\}$ 在乘法下构成群：
- **单位元：** 数 $1$ 满足 $1 \cdot a = a \cdot 1 = a$
- **逆元：** 对每个 $a \in \mathbb{Q}^*$，数 $1/a$ 满足 $a \cdot (1/a) = 1$

注意我们必须排除 $0$，因为它没有乘法逆元。
</Example>

<Example number="1.4.4" name="正实数">
集合 $\mathbb{R}^+ = \{x \in \mathbb{R} : x > 0\}$ 在乘法下构成阿贝尔群。
</Example>

### 模运算

<Example number="1.4.5" name="模 n 整数">
对任意正整数 $n$，集合 $\mathbb{Z}/n\mathbb{Z} = \{0, 1, 2, \ldots, n-1\}$ 在模 $n$ 加法下构成群：
- **单位元：** $0$
- **$a$ 的逆元：** $n - a$（当 $a \neq 0$ 时）

这是阶为 $n$ 的有限阿贝尔群。
</Example>

<Example number="1.4.6" name="模 n 单位群">
集合 $(\mathbb{Z}/n\mathbb{Z})^* = \{a \in \mathbb{Z}/n\mathbb{Z} : \gcd(a, n) = 1\}$ 在模 $n$ 乘法下构成群。这个群的阶是 $\phi(n)$，即欧拉函数。
</Example>

### 矩阵群

<Example number="1.4.7" name="一般线性群">
**一般线性群** $\text{GL}_n(\mathbb{R})$ 由所有实数项的 $n \times n$ 可逆矩阵组成，运算是矩阵乘法：
$$\text{GL}_n(\mathbb{R}) = \{A \in M_n(\mathbb{R}) : \det(A) \neq 0\}$$
当 $n \geq 2$ 时这是非阿贝尔群。
</Example>

<Example number="1.4.8" name="特殊线性群">
**特殊线性群** $\text{SL}_n(\mathbb{R}) = \{A \in \text{GL}_n(\mathbb{R}) : \det(A) = 1\}$ 是 $\text{GL}_n(\mathbb{R})$ 的子群。
</Example>

### 对称群

<Example number="1.4.9" name="对称群">
**对称群** $S_n$ 由 $\{1, 2, \ldots, n\}$ 到自身的所有双射组成，运算是函数复合。这些双射称为**置换**。

$S_n$ 的阶是 $n!$（阶乘）。当 $n \geq 3$ 时，群 $S_n$ 是非阿贝尔的。
</Example>

<Example number="1.4.10" name="二面体群">
**二面体群** $D_n$ 是正 $n$ 边形的对称群，包括旋转和反射。它的阶是 $2n$，当 $n \geq 3$ 时是非阿贝尔的。
</Example>

## 1.5 子群

*子群*是群的一个子集，它本身在继承的运算下也是群。子群对于理解群的内部结构至关重要。

<Definition number="1.5.1" name="子群">
设 $(G, \star)$ 是群。子集 $H \subseteq G$ 是 $G$ 的**子群**，记作 $H \leq G$，若 $(H, \star)$ 在同一运算下本身是群。
</Definition>

<Theorem number="1.5.2" name="子群判定准则">
设 $G$ 是群，$H$ 是 $G$ 的非空子集。则 $H$ 是 $G$ 的子群当且仅当：
1. 对所有 $a, b \in H$，有 $ab \in H$（封闭性）
2. 对所有 $a \in H$，有 $a^{-1} \in H$（逆元封闭性）
</Theorem>

<Proof>
($\Rightarrow$) 若 $H$ 是子群，则它是群，所以它必须对运算和取逆封闭。

($\Leftarrow$) 设 $H$ 满足条件 (1) 和 (2)。我们验证群公理：
- **结合律：** 从 $G$ 继承
- **单位元：** 因为 $H$ 非空，存在 $a \in H$。由 (2)，$a^{-1} \in H$。由 (1)，$e = a \cdot a^{-1} \in H$。
- **逆元：** 由条件 (2) 给出
- **封闭性：** 由条件 (1) 给出
</Proof>

<Theorem number="1.5.3" name="一步子群判定法">
设 $G$ 是群，$H$ 是 $G$ 的非空子集。则 $H \leq G$ 当且仅当对所有 $a, b \in H$，有 $ab^{-1} \in H$。
</Theorem>

<Example number="1.5.4" name="平凡子群">
每个群 $G$ 至少有两个子群：
1. **平凡子群** $\{e\}$
2. **非真子群** $G$ 本身

满足 $\{e\} \subsetneq H \subsetneq G$ 的子群 $H$ 称为**非平凡真子群**。
</Example>

<Example number="1.5.5" name="整数的子群">
$(\mathbb{Z}, +)$ 的子群恰好是集合 $n\mathbb{Z} = \{nk : k \in \mathbb{Z}\}$，其中 $n \geq 0$。因此：
$$\{0\} \subsetneq \cdots \subsetneq 6\mathbb{Z} \subsetneq 3\mathbb{Z} \subsetneq \mathbb{Z}$$
</Example>

## 1.6 群同态

*同态*是群之间保持群结构的函数。同态是群论中的"保结构映射"。

<Definition number="1.6.1" name="群同态">
设 $(G, \star)$ 和 $(H, \cdot)$ 是群。函数 $\phi : G \to H$ 是**同态**，若对所有 $a, b \in G$：
$$\phi(a \star b) = \phi(a) \cdot \phi(b)$$
</Definition>

这个条件说 $\phi$ "保持"或"尊重"群运算：先在 $G$ 中组合元素再应用 $\phi$，与先应用 $\phi$ 再在 $H$ 中组合，结果相同。

<Theorem number="1.6.2" name="同态的性质">
设 $\phi : G \to H$ 是群同态。则：
1. $\phi(e_G) = e_H$
2. 对所有 $a \in G$，$\phi(a^{-1}) = \phi(a)^{-1}$
3. 对所有 $a \in G$ 和 $n \in \mathbb{Z}$，$\phi(a^n) = \phi(a)^n$
</Theorem>

<Proof>
(1) 我们有 $\phi(e_G) = \phi(e_G \cdot e_G) = \phi(e_G) \cdot \phi(e_G)$。两边乘以 $\phi(e_G)^{-1}$ 得 $e_H = \phi(e_G)$。

(2) 我们有 $\phi(a) \cdot \phi(a^{-1}) = \phi(a \cdot a^{-1}) = \phi(e_G) = e_H$。因此 $\phi(a^{-1})$ 是 $\phi(a)$ 的逆元。

(3) 由 (1) 和 (2) 归纳得出。
</Proof>

<Definition number="1.6.3" name="核与像">
设 $\phi : G \to H$ 是同态。

$\phi$ 的**核**是：
$$\ker(\phi) = \{a \in G : \phi(a) = e_H\}$$

$\phi$ 的**像**是：
$$\text{im}(\phi) = \{\phi(a) : a \in G\} = \phi(G)$$
</Definition>

<Theorem number="1.6.4">
设 $\phi : G \to H$ 是同态。则：
1. $\ker(\phi)$ 是 $G$ 的子群
2. $\text{im}(\phi)$ 是 $H$ 的子群
</Theorem>

<Definition number="1.6.5" name="同态的类型">
同态 $\phi : G \to H$ 称为：
- **单同态**（或**单射**），若 $\phi$ 是单射
- **满同态**（或**满射**），若 $\phi$ 是满射
- **同构**，若 $\phi$ 是双射
- **自同构**，若 $\phi : G \to G$ 是同构
</Definition>

<Theorem number="1.6.6">
同态 $\phi : G \to H$ 是单射当且仅当 $\ker(\phi) = \{e_G\}$。
</Theorem>

<Proof>
($\Rightarrow$) 若 $\phi$ 是单射且 $\phi(a) = e_H$，则 $\phi(a) = \phi(e_G)$，所以 $a = e_G$。

($\Leftarrow$) 若 $\ker(\phi) = \{e_G\}$ 且 $\phi(a) = \phi(b)$，则 $\phi(ab^{-1}) = \phi(a)\phi(b)^{-1} = e_H$，所以 $ab^{-1} \in \ker(\phi) = \{e_G\}$，得 $a = b$。
</Proof>

<Definition number="1.6.7" name="同构的群">
两个群 $G$ 和 $H$ **同构**，记作 $G \cong H$，若存在同构 $\phi : G \to H$。
</Definition>

同构的群从代数角度看"本质上相同"——它们有完全相同的群论性质，只是元素的名称不同。

## 1.7 循环群

在所有群中，循环群是最简单、理解最透彻的。尽管简单，它们在理论中起着核心作用。

<Definition number="1.7.1" name="循环群">
群 $G$ 是**循环的**，若存在元素 $g \in G$ 使得 $G$ 的每个元素都可以写成 $g$ 的幂：
$$G = \{g^n : n \in \mathbb{Z}\} = \langle g \rangle$$
元素 $g$ 称为 $G$ 的**生成元**。
</Definition>

<Example number="1.7.2">
群 $(\mathbb{Z}, +)$ 是循环群，生成元是 $1$（或 $-1$）。每个整数都是 $1$ 在加法下的"幂"：$n = 1 + 1 + \cdots + 1$（$n$ 次）。
</Example>

<Example number="1.7.3">
群 $\mathbb{Z}/n\mathbb{Z}$ 是循环群，生成元是 $1$。它是阶为 $n$ 的唯一循环群（在同构意义下）。
</Example>

<Theorem number="1.7.4" name="循环群的分类">
每个循环群同构于 $\mathbb{Z}$（若是无限的）或 $\mathbb{Z}/n\mathbb{Z}$（若是阶为 $n$ 的有限群）。
</Theorem>

<Theorem number="1.7.5">
循环群的每个子群都是循环的。
</Theorem>

<Theorem number="1.7.6" name="有限循环群的子群">
设 $G = \langle g \rangle$ 是阶为 $n$ 的循环群。则：
1. 对 $n$ 的每个因子 $d$，恰好存在一个阶为 $d$ 的子群
2. 这个子群是 $\langle g^{n/d} \rangle$
</Theorem>

## 总结

<Note>
本章建立了群论的基础：

1. **群**是具有结合二元运算、单位元和所有元素的逆元的集合

2. **基本性质**包括单位元和逆元的唯一性、消去律以及公式 $(ab)^{-1} = b^{-1}a^{-1}$

3. **例子**从加法或乘法下的数系，到矩阵群，到几何对象的对称群

4. **子群**是在继承运算下本身也是群的子集

5. **同态**是群之间的保结构映射；同构是双射同态

6. **循环群**由单个元素生成，已被完全分类
</Note>

这里发展的理论为后续所有章节提供了基础。在第二章中，我们将学习*陪集*和*拉格朗日定理*，它们揭示了群与其子群之间的深刻联系。
